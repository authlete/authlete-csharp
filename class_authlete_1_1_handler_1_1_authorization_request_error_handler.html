Detailed Description

Handler for error cases of authorization requests.

A response from Authlete's /api/auth/authorization API contains an "action" response parameter. When the value of the response parameter is neither "NO_INTERACTION" nor "INTERACTION", the authorization request should be handled as an error case. This class is a handler for such error cases.

HttpResponseMessage Handle ( AuthorizationResponse  response)
inline

Handle an error case of an authorization request. This method returns null when response.Action returns AuthorizationAction.INTERACTION or AuthorizationAction.NO_INTERACTION. In other cases, an instance of HttpResponseMessage is returned.

Returns
An error response that should be returned to the client application from the authorization endpoint. null is returned when response.Action returns AuthorizationAction.INTERACTION or AuthorizationAction.NO_INTERACTION.
Parameters
responseA response from Authlete's /api/auth/authorization API.
