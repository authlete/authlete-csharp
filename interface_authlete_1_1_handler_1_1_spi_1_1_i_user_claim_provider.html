Get the value of a claim of the user. This method may be called multiple times.

Returns
The value of the claim. null if the value is not available. In most cases, a string should be returned. When claimName is "address", an instance of Address class should be returned.
Parameters
subjectThe subject (= unique identifier) of a user.
Parameters
claimNameA claim name such as "name" and "family_name". Standard claim names are listed in 5.1. Standard Claims of OpenID Connect Core 1.0. Constant values that represent the standard claims are listed in Authlete.Types.StandardClaims class. Note that the value of this argument (claimName) does NOT contain a language tag.
Parameters
languageTagA language tag such as "en" and "ja". Implementations of this method should take this into consideration if possible. See 5.2. Claims Languages and Scripts of OpenID Connect Core 1.0 for details.

Implemented in UserClaimProviderAdapter.
